# Hujan Ga

it using laravel 12

requirement

-   php 8.2^
-   nodeJs
-   PC

library

-   chartJs
-   openlayer
-   kreait/firebase-php
-   tailwindcss

## How to install

clone this repo

```bash
$ git clone https://github.com/FaiqSr/hujanga
```

after that

you can install the library

```bash
$ composer install
```

and

```bash
$ npm install
```

## How to run

make 2 terminal

first terminal run this script

```bash
$ php artisan serve
```

second terminal run this script

```bash
$ npm run dev
```
